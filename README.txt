Foodle: Manuel d'instruction

Avant toute chose, il est nécessaire d'installer un server local sur votre machine en saisissant la ligne de commande *python -m http.server 8383*  en se positionnant dans le répertoire source de l'archive fournie.

Ensuite, lancez un navigateur sur localhost:8383/index.html (IMPORTANT - il faut que l'url contienne explicitement index.html)

Pour commencer une recherche, rendez-vous sur la page d'accueil (index.html) puis choisissez le type de l'élément que vous souhaitez rechercher parmi les onglets "Search by".

1) Effectuer une recherche
Pour rechercher un plat par son nom, ses ingrédients ou son type, choisissez respectivement l'onglet Name, Ingredient ou Type puis tapez le nom du sujet de votre recherche. Vous pouvez choisir de filtrer les résultats en fonction du pays d'origine du plat en indiquant son nom dans le champ "Filter by country". Une liste des pays et zones géographiques existant sur DBpedia s'affiche dans un menu déroulant, vous pouvez en sélectionner un dans la liste. 
Pour directement rechercher un pays, choisissez l'onglet Country et tapez son nom / sélectionnez le dans la liste des pays proposés.

2) Highlighted Food
Cette section de la page présente un nouveau plat à chaque chargement et possède un lien cliquable pour accéder à la page du plat concerné.

3) Page résultats de recherche
Les résultats de votre recherche de plat s'affichent dans des sections cliquables et vous redirigeant vers la page plat correspondant à la description, image et nom présentés. Dans le cas d'une recherche par ingrédient le bouton "Buy at Walmart" apparaît vous permettant d'être redirigés vers le site de Walmart pour acheter l'ingrédient en question.

4) Page détail plat
Cette page présente les informations générales d'un plat. Elle permet aussi d'accéder aux pages des plats similaires, pays ou zones géographiques d'origine, ingrédients ou type de plat reliés au sujet de la page, par liens cliquables. Les plats similaires s'affichent dans une galerie dynamique d'images qu'il est possible de faire défiler en appuyant sur les flèches.

5) Page détail pays
Le nom, la description et le drapeau du pays s'affichent, suivis de la liste de ses plats caractéristiques avec encore photographie  et description. Il suffit de cliquer sur la section pour accéder à la page plat correspondante.
Il est possible de filtrer les plats du pays en tapant le nom du plat cherché dans le champ "Filter results". Remontent alors le ou les résultats concerné(s).

6) Quick search
Il est possible depuis n'importe quelle autre page que index d'effectuer une recherche simple de plat en tapant son nom dans la barre de recherche "Quick Search" en haut à droite.

7) Retour à la page d'accueil
Il est possible de retourner à la page d'accueil depuis n'importe quelle page en cliquant sur le logo Foodle.
